# This is Small Hotel example with following chracteristics:

Important technologies used in this example:

- ASP.NET Core and C# for cross-platform server-side code
- Angular and TypeScript for client-side code
- Webpack for building and bundling client-side resources
- Bootstrap for layout and styling
- MS SQL Server Management Studio for database
- Entity Framework Core models including database context class
- Manipulating Data Using LINQ
- Web Services using ASP.NET Core and JSON serialization
